- **AGENTS.md**: General project guidance for agents and contributors
- **CLAUDE.md**: Lightweight pointer for Claude-based harnesses
- **PENDING_TASKS.md**: Small, pickable follow-up work items
- **COMPLETED_TASKS.md**: Implemented project capabilities and milestones
- **IMPLEMENTATION.md**: Architecture, UI behavior, and technical details
- **PROJECT_PLAN.md**: Current project direction and priorities
- **README.md**: Human-facing setup and usage guide
- **TEST.md**: Automated and manual testing guidance for this repository

Use `IMPLEMENTATION.md` for environment setup, structure, conventions, and
architecture details. Use `TEST.md` for automated and manual testing guidance.
